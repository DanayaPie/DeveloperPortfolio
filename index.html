Projects

Sweet Lime - Coming Soon

Sweet Lime is an upcoming skincare price tracking web application designed to help users make informed purchasing decisions by providing insights into price history. Users will be able to input product URLs from Stylevana.com to view historical price trends and determine the best time to buy.

The application will feature user login capabilities with a favorites option, price drop notifications, and support for additional websites such as YesStyle, Olive Young, and more in future updates.

Fortune Hangman

Fortune Hangman is a dynamic multiplayer word/phrase-guessing game that challenges players to achieve the highest score across multiple rounds. The game is available in two formats: Command Line and web-based User Interface. The Command Line version offers a classic, text-based gameplay experience with a spinning wheel that introduces elements of chance, including points, bankruptcy, losing a turn, and spin tokens. The web-based User Interface version enhances the experience with a visually interactive design and will include the spinning wheel feature in future updates, providing a more engaging and immersive gameplay experience.
